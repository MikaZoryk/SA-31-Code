Patient Portal

“Patient Portal” - це інноваційна система прийняття рішень в галузі медичної генетики, яка забезпечує ефективне використання генетичних даних для покращення діагностики та лікування хвороб.

About

“Patient Portal” - це важливий крок вперед у використанні генетики для покращення здоров’я людей. Ця система відкриває нові можливості для розуміння, діагностики та лікування генетичних хвороб
